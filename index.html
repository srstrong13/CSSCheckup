Classes

A CSS class is an identifier of an element which can be used multiple times on a page.

Memory device:Classes in school are periods (first and second period, etc.) so remeber to use a . whenever you want to reference a class in your CSS.

  • Classes start with a . in the selector .anything-you-want-to-name-it
  • The same class name can be used more than once within the same file
  • The class keyword refers to the class name that you set in the HTML page.
